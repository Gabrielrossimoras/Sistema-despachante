# Sistema-despachante
Modelagem de sistema e banco de dados para controle de documentos de despachante.
1 - Descrição do sistema
O sistema tem como objetivo auxiliar o despachante no controle de documentos e processos de veículos, evitando atrasos, perda de prazos e falta de organização das informações. O sistema será utilizado pelo despachante para cadastrar clientes, veículos, origem dos atendimentos, tipos de serviços, documentos necessários, prazos e status de cada processo.
A proposta do sistema é permitir que o despachante acompanhe cada atendimento desde o cadastro inicial até a finalização do processo. Com isso, será possível consultar processos em andamento, identificar documentos pendentes, controlar prazos e reduzir falhas no acompanhamento dos serviços prestados.
2 - Problema resolvido pelo sistema
Atualmente, muitos processos de documentação podem sofrer atrasos por falta de controle adequado dos prazos, documentos pendentes ou informações espalhadas. O sistema busca resolver esse problema centralizando os dados dos clientes, veículos e processos em um único ambiente, facilitando a organização e a tomada de decisão do despachante.
3 - Usuário principal
O usuário principal do sistema será o despachante, responsável por cadastrar, consultar, acompanhar e atualizar os processos relacionados à documentação de veículos.
4 - Funcionalidades principais
•	Cadastrar clientes; 
•	Cadastrar veículos; 
•	Cadastrar origem do atendimento; 
•	Cadastrar tipos de serviço; 
•	Registrar processos; 
•	Controlar documentos pendentes; 
•	Controlar prazos; 
•	Atualizar status do processo; 
•	Consultar processos em andamento, atrasados e finalizados.
